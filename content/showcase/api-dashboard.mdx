这是一个使用 Pivot 组件库构建的 API 管理仪表板示例。展示了如何使用各种组件来创建一个功能完整的 API 文档和测试界面。

## 主要特性

- **端点总览** - 清晰显示所有 API 端点
- **实时状态** - 显示 API 的健康状态和响应时间
- **交互式文档** - 支持直接在界面中测试 API
- **多种认证** - 支持多种认证方式
- **响应式设计** - 完美适配各种设备

## 使用的组件

- `StatusCode` - 显示 HTTP 状态码
- `MethodLabel` - HTTP 方法标签
- `TypeIndicator` - 数据类型指示器
- `Card` - 卡片布局
- `Badge` - 状态徽章
- `Button` - 操作按钮

## 技术栈

- React 18
- Next.js 15
- Tailwind CSS
- Pivot UI Components
- TypeScript
