用于显示联系信息的组件，包括邮箱、网站等。

<ComponentPreview name="contact-display-demo" />

## 安装

```bash
npx shadcn@latest add https://pivotkit.vercel.app/registry.json contact-display
```

## 使用方法

```tsx
import { ContactDisplay } from "@/components/pivot/contact-display";
```

```tsx
<ContactDisplay />
```
