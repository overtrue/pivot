## 安装

```bash
npx shadcn@latest add https://pivotkit.vercel.app/r/typescript-generator.json
```

## 使用方法

```tsx
import { TypeScriptGenerator } from "@/components/pivot/typescript-generator";
```

```tsx
<TypeScriptGenerator />
```
