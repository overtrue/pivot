External Docs Display 组件用于显示外部文档的详细信息，包括链接和描述。

<ComponentPreview name="external-docs-display-demo" />

## 安装

```bash
npx shadcn@latest add https://pivotkit.vercel.app/registry.json external-docs-display
```

## 使用方法

```tsx
import { ExternalDocsDisplay } from "@/components/pivot/external-docs-display";
```

```tsx
<ExternalDocsDisplay />
```

## 示例

### 基本用法

```tsx
<ExternalDocsDisplay />
```

## API 参考

### Props
