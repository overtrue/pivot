## 📚 Hooks 和工具

除了组件，Pivot 还提供了强大的 hooks 和工具函数：

### 🎣 React Hooks

```tsx
import { useOpenApi } from "@/registry/lib/hooks/use-openapi";
import { useOperation } from "@/registry/lib/hooks/use-operation";
import { useSchema } from "@/registry/lib/hooks/use-schema";
```

### 🛠️ 工具函数

```tsx
import { resolveRef } from "@/registry/lib/utils/resolve-ref";
import { generateExample } from "@/registry/lib/utils/generate-example";
import { isReferenceObject } from "@/registry/lib/utils/schema-utils";
```

### 🌍 国际化

```tsx
import { useI18n, t } from "@/registry/lib/i18n";
```

## 🚀 快速安装

### 单个组件

```bash
npx shadcn@latest add https://pivotkit.vercel.app/registry.json status-code
```

### 工具函数

```bash
npx shadcn@latest add https://pivotkit.vercel.app/registry.json hooks
npx shadcn@latest add https://pivotkit.vercel.app/registry.json resolve-ref
```
