这里是所有可用组件的列表。每个组件都经过精心设计，可以直接复制到您的项目中使用。

## 基础组件

这些是构建界面的基础组件：

- [Button](/docs/components/button) - 按钮组件，支持多种变体和大小
- [Card](/docs/components/card) - 卡片容器组件
- [Badge](/docs/components/badge) - 徽章组件，用于状态显示
- [Input](/docs/components/input) - 输入框组件
- [Textarea](/docs/components/textarea) - 文本域组件

## 布局组件

用于构建页面布局的组件：

- [Container](/docs/components/container) - 容器组件，用于页面布局
- [Grid](/docs/components/grid) - 网格布局组件
- [Stack](/docs/components/stack) - 堆叠布局组件

## 数据展示

用于展示数据的组件：

- [Table](/docs/components/table) - 表格组件
- [List](/docs/components/list) - 列表组件
- [Tooltip](/docs/components/tooltip) - 提示框组件

## 反馈组件

用于用户反馈的组件：

- [Alert](/docs/components/alert) - 警告提示组件
- [Toast](/docs/components/toast) - 消息提示组件
- [Modal](/docs/components/modal) - 模态框组件

## OpenAPI 专用组件

专门为 OpenAPI 文档设计的组件：

- [API Explorer](/docs/components/api-explorer) - API 接口浏览器
- [Schema Viewer](/docs/components/schema-viewer) - 数据模型查看器
- [Code Block](/docs/components/code-block) - 代码块组件
- [Response Viewer](/docs/components/response-viewer) - 响应结果查看器

## 特殊效果

增强用户体验的动效组件：

- [Magic Card](/docs/components/magic-card) - 魔法卡片效果
- [Shimmer](/docs/components/shimmer) - 闪光效果
- [Ripple](/docs/components/ripple) - 水波纹效果

## 使用指南

每个组件页面都包含：

1. **安装说明** - 如何安装和配置组件
2. **API 文档** - 完整的属性和方法说明
3. **使用示例** - 多种使用场景的代码示例
4. **最佳实践** - 使用建议和注意事项
