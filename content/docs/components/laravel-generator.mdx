用于生成 Laravel 代码的组件。

<ComponentPreview name="laravel-generator-demo" />

## 安装

```bash
npx shadcn@latest add https://pivotkit.vercel.app/registry.json laravel-generator
```

## 使用方法

```tsx
import { LaravelGenerator } from "@/components/pivot/laravel-generator";
```

```tsx
<LaravelGenerator />
```

## 示例

### 基本用法

```tsx
<LaravelGenerator />
```
