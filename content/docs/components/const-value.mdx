用于显示常量值的组件，支持不同类型的常量展示。

<ComponentPreview name="const-value-demo" />

## 安装

```bash
npx shadcn@latest add https://pivotkit.vercel.app/registry.json const-value
```

## 使用方法

```tsx
import { ConstValue } from "@/components/pivot/const-value";
```

```tsx
<ConstValue />
```

## 示例

### 基本用法

```tsx
<ConstValue />
```
