我们很高兴地宣布 Pivot 的正式发布！Pivot 是一个专门为 OpenAPI 规范设计的现代化 React UI 组件库。

## 为什么需要 Pivot？

在构建 API 文档时，开发者经常面临以下挑战：

1. **设计不一致** - 不同的组件使用不同的设计语言
2. **缺乏专业性** - 通用组件无法很好地展示 API 特有的信息
3. **开发效率低** - 需要从头开始构建每个组件

Pivot 解决了这些问题，提供了一套专门为 API 文档设计的组件。

## 核心特性

### 🎯 专门优化

每个组件都是专门为 OpenAPI 规范设计的，能够完美展示：

- HTTP 状态码
- 请求方法
- 数据类型
- 参数信息
- 响应格式

### 🎨 现代化设计

基于 Tailwind CSS 和现代设计原则，提供：

- 清晰的视觉层次
- 一致的颜色编码
- 优雅的动画效果
- 完美的暗色模式支持

### 🚀 开发者友好

采用 shadcn/ui 的 registry 系统：

- 复制粘贴即可使用
- 完整的 TypeScript 支持
- 灵活的定制选项
- 详细的文档和示例

## 快速开始

安装 Pivot 非常简单：

```bash
npx create-next-app@latest my-app --typescript --tailwind
cd my-app
npx shadcn@latest add https://pivotkit.vercel.app/registry.json status-code
```

然后就可以在项目中使用了：

```tsx
import { StatusCode } from "@/components/pivot/status-code";

export function ApiResponse() {
  return (
    <div className="flex items-center gap-2">
      <span>响应状态：</span>
      <StatusCode code={200} />
    </div>
  );
}
```
